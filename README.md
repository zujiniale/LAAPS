## Como compilar e rodar

```bash
javac GuiaSegurancaDigital.java
java GuiaSegurancaDigital
```

## Requisitos atendidos

| Requisito       | Onde no código                          |
|-----------------|-----------------------------------------|
| `while`         | Loop principal do menu (linha 10)       |
| `switch`        | Navegação entre módulos (linha 22)      |
| `if`            | Avaliação da senha e resposta do golpe  |
| `Scanner`       | Leitura de todas as entradas do usuário |
| Menu interativo | Usuário navega e retorna ao menu        |
| 2+ tópicos      | Módulo 1: Senhas · Módulo 2: Golpes     |

## Estrutura

- **1 arquivo:** `GuiaSegurancaDigital.java`
- **~80 linhas** de código
